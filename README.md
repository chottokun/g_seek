# Research Assistant 🔬🚀

AIを活用した次世代型リサーチ自動化ツールです。  
LLM（大規模言語モデル）とWeb検索エンジンを最適に組み合わせ、指定されたトピックに対して「計画立案 → 検索 → 評価 → 統合」のサイクルを自律的に回し、構造化された詳細レポートを生成します。

## 主な特徴 🛠️

- **Gemini / OpenAI / Ollama 対応**: 最新の LLM を活用。Gemini API 等の多様なレスポンス形式に対する堅牢なパース機能を搭載。
- **Streamlit 推奨 UI**: 高い安定性と多機能性を備えたメインインターフェース。
  - **確実な成果物提供**: レポート (Markdown) やネットワーク図 (HTML) の直接ダウンロード。
  - **インライン編集**: AI が提案したリサーチ計画をその場で修正・承認可能。
  - **リアルタイム進捗**: スクロールログとプログレスバーにより、並列処理の状況を可視化。
- **動的スキルの学習と活用**: 過去のリサーチから抽出された専門知見を「スキル」として蓄積。JSON 形式の推論プロセスを経て、現在の調査に最適なスキルを厳格に選択（無関係なデータの混入を防止）。
- **uv による高速管理**: 最新の `uv` を活用し、依存関係の不整合を排除した超高速なセットアップを実現。

## セットアップ

### 1. `uv` を使用した環境構築

本プロジェクトでは [uv](https://github.com/astral-sh/uv) を使用したパッケージ管理を推奨しています。

```bash
# 依存関係のインストール
uv sync

# 仮想環境の有効化（オプション）
source .venv/bin/activate
```

### 2. 環境変数の設定

`.env` ファイルを作成し、必要な API キーを設定してください。

```bash
cp .env.example .env
```

主な設定項目:
- `LLM_PROVIDER`: `gemini`, `openai`, `azure_openai`, `ollama` 等
- `SEARCH_API`: `tavily`, `searxng`, `duckduckgo` 等
- `EVOLVE_SKILLS`: リサーチを通じた動的スキルの学習を有効にするか (`true` / `false`)

## 使い方

### Streamlit UI (推奨 🌟)
最も安定しており、レポートのダウンロードや計画の編集が容易なメイン UI です。

```bash
uv run streamlit run deep_research_project/streamlit_app.py
```
デフォルトで http://localhost:8501 でアクセス可能です。

### Chainlit UI (参考実装)
チャット形式の UI です。※特定のブラウザ環境でファイルのダウンロードに制限が発生する場合があります。

```bash
uv run chainlit run deep_research_project/chainlit_app.py
```
デフォルトで http://localhost:8000 でアクセス可能です。

### CLI (テスト用)
コマンドラインから直接リサーチを実行できます。

```bash
# 基本実行
uv run -m deep_research_project.main "調査テーマ"

# 詳細オプション指定
uv run -m deep_research_project.main "テーマ" --loops 5 --results 5 --snippets
```

## Docker 実行

Docker Compose を使用して、アプリ単体または SearXNG/Ollama を含めたフルスタック環境を起動できます。

```bash
# 基本起動
docker compose up --build

# SearXNG や Ollama と連携する場合
docker compose -f docker-compose.yaml -f docker-compose.searxng.yaml -f docker-compose.ollama.yaml up --build
```
